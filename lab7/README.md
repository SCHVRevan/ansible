# Lab_7

## Постановка задачи
Установите PostgreSQL на своей виртуальной машине машине  
Измените порт по умолчанию на 5452  
Создайте новую базу данных secure_db  
Создайте:
- Роль-группу readers с правом только на SELECT для всех таблиц в схеме public
- Роль-пользователя user01, включите его в группу readers
- Отзовите право на создание объектов в схеме public у роли PUBLIC

## Описание
К основному заданию добавил минимальное логирование, настройку параметров безопасности (Настройки безопасности. Резюме по настройкам безопасности), а также добавление скрипта для создания резервных копий раз в сутки в ночное время.
