# Lab_2.1

## Постановка задачи
Создать playbook который собирает указанную информацию о хостах:  
- Хост
- FDQN
- Версия
- Конфиг
- Дата/время
- Пользователь
- Версия ОС
- Семейство ОС
- RAM
- ip адрес
- Время работы: X дней X часов X минут

## Описание
Отключил сбор информации в конфигурации для данного проекта. Для сбора информации использовал модуль setup с методом gather_subset.  
Inventory разделил по подгруппам для удобства запуска playbook'a как по заданным группам (local or remote), так и по отдельным хостам по их имени (заданном в inventory).

## Команды
```bash
ansible-playbook get_info.yml > output.txt
```
