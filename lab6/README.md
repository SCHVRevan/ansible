# Lab_6

## Постановка задачи
### Установка, конфигурация, написание правил  
Необходимо установить и настроить службу auditd. В качестве обязательных настроек, должны присутствовать:
- аудит действий пользователя с повышенными привилегиями
- аудит изменения конфигурационного файла /etc/ssh/sshd_config
- аудит выполнения команд chmod и chown

Выполните команду
```bash
sudo auditctl -l > report1.txt
```
В правила добавьте ключ key, чтобы в дальнейшем было удобно искать нужные команды.

### Проверка работы написанных правил  
Выполнить подряд три действия:
- Запустить команду с повышенными привилегиями (например sudo ls)
- Поправить конфигурационный файл /etc/ssh/sshd_config, можно добавить строчку с комментарием
- Выполнить chmod +r <любой_файл>

Если аудит настроен и работает корректно, информация о выполнении данных команд, должна попасть в файл /var/log/audit/audit.log.
