## Вспомогательные плейбуки для настройки целевого узла

Перед выполнением плейбуков убедитесь, что:
- **чистый** тестовый диск примонтирован к виртуальной машине  
- под него выделено по крайней мере 10G  
- диск является вторым в иерархии примонтированных дисков (так как плейбуки захаржены под работу с sdb)  

После отработки плейбука систему требуется перезагрузить для применения изменений из fstab.

### Команды
```bash
ansible-playbook pb_new_disk_processing.yml --ask-become-pass
```
Если разметка и монтирование уже выполнены, можно использовать исключительно группу задач для конфигурирования прав доступа
```bash
ansible-playbook pb_new_disk_processing.yml --tags "permissions" --ask-become-pass
```
